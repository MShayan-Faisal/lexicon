📚 Lexicon – The Smart Dictionary App

Lexicon is a modern, responsive dictionary web application built using HTML, CSS, and JavaScript. It allows users to search for words and instantly view their definitions, pronunciations, parts of speech, and example usages — all in a clean, elegant interface.

This project combines functional logic with appealing design, creating a seamless learning tool for anyone who loves exploring language.

About The Project

Lexicon was developed to make looking up words simple, fast, and enjoyable. Instead of using cluttered dictionary sites, I wanted to design a lightweight app that focuses purely on what matters — meaning, clarity, and usability.

The app connects to a free dictionary API to fetch accurate, real-time definitions. It also provides pronunciation and examples to help users understand the word in context.

Every part of Lexicon was built with usability and accessibility in mind — from the search bar to the dynamic content display.

Technologies Used

How It Works

The user types a word into the search bar.

JavaScript sends a request to the Dictionary API.

The API returns detailed information about the word — including definition, pronunciation, and usage.

The data is displayed dynamically on the page without reloading.

If the word isn’t found, an error message appears guiding the user to try another term.
